# Income Calculator Web App

This repository contains a simple web application that allows users to calculate their income by adding and removing individual incomes to obtain the total. This Readme file provides an overview of the project, installation instructions, usage guidelines, and other relevant information.

## Live Demo

You can try out the live demo of the Income Calculator Web App by visiting the following link: [Income Calculator Web App Demo](https://jobvillagran.github.io/calculadoraIngresos/)

## Features

- Add and remove incomes: Users can enter the name and amount of each income source and add it to the calculation. They can also remove incomes if needed.
- Calculate total income: The web app dynamically calculates the total income based on the added incomes.
- Clear all incomes: Users can clear the list of incomes and start fresh.

## Technologies Used

- HTML: Markup language for creating the structure and elements of the web app.
- CSS: Stylesheet language for enhancing the visual appearance and layout.
- JavaScript: Programming language used for implementing the interactivity and calculations.

## Installation

To run the Income Calculator Web App locally, follow these steps:

1. Clone the repository to your local machine using the following command:
   ```
   git clone https://github.com/JobVillagran/calculadoraIngresos.git
   ```

2. Navigate to the project directory:
   ```
   cd calculadoraIngresos
   ```

3. Open the `index.html` file in your preferred web browser.

That's it! The web app should now be up and running on your local machine.

## Usage

1. Open the web app in your web browser.
2. Enter the name and amount of an income in the respective input fields.
3. Click the "Add Income" button to add the income to the calculation.
4. The total income will be updated dynamically as incomes are added or removed.
5. To remove an income, click the "Remove" button next to the income in the list.
6. To clear all incomes and start fresh, click the "Clear All" button.
